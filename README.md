# code-graph

Default Electron + React + TypeScript starter application.

## Requirements

- Node.js 20+
- npm 10+

## Install

```bash
npm install
```

## Run in development

```bash
npm run dev
```

Electron main/preload are authored in TypeScript and transpiled to the `out/` folder before Electron launches.

## Build

```bash
npm run build
```

## Start built app

```bash
npm run start
```
